Minipaca is a tool to build cross platform apps with Chez Scheme and GTK4.
More information: https://minipaca.github.io

To run minipaca you must have Chez Scheme installed on your operating system. Here is howtos:

- *Debian 12, Ubuntu 22.04, Ubuntu 24.04 (x86_64/amd64)*
Chez Scheme is in the official repos, it will be installed automatically

- *Arch Linux (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-cs10arch-x86_64.pkg.tar.zst][Chez Scheme 10.0.0]] and install it
#+begin_src sh
$ sudo pacman -U ./xx.pkg.tar.zst
#+end_src

- *Arch Linux (aarch64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-aarch64cs10arch-aarch64.pkg.tar.zst][Chez Scheme 10.0.0]] and install it
#+begin_src sh
$ sudo pacman -U ./xx.pkg.tar.zst
#+end_src

- *Fedora Linux (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-cs10fc-x86_64.rpm][Chez Scheme 10.0.0]] and install it
#+begin_src sh
$ sudo dnf install ./xx.rpm
#+end_src

- *openSUSE Linux (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-cs10opensuse-x86_64.rpm][Chez Scheme 10.0.0]] and install it
#+begin_src sh
$ sudo zypper install ./xx.rpm
#+end_src

- *Void Linux (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0_7.x86_64.xbps][Chez Scheme 10.0.0]] and install it
#+begin_src sh
$ xbps-rindex -a xx-version.x86_64.xbps
$ sudo xbps-install -R ./ xx-version
#+end_src

- *FreeBSD (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0.pkg][Chez Scheme 10.0.0]] and install it
#+begin_src sh
pkg add -f xx.pkg    # As root
#+end_src

- *NetBSD (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0.tgz][Chez Scheme 10.0.0]] and install it
#+begin_src sh
pkg_add ./xx.tgz    # As root
#+end_src

- *macOS (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-cs10macos-amd64.pkg][Chez Scheme 10.0.0]] and install it

- *Windows OS (x86_64/amd64)*
Download [[https://raw.githubusercontent.com/minipaca/minipaca/master/chez-scheme-10.0.0/chez-scheme-10.0.0-10.0.0-7-cs10win-amd64.exe][Chez Scheme 10.0.0]] and install it
Download [[https://aka.ms/vs/17/release/VC_redist.x64.exe][VC_redist]] and install it
