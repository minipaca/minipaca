Minipaca is a tool to build cross platform apps with Chez Scheme and GTK4.
More information: https://minipaca.github.io

To run minipaca you must have Chez Scheme installed on your operating system. Here is howtos:

- For Ubuntu 22.04, Debian 12, LDME 6 (x86_64/amd64):
Chez Scheme is in the official repo, it will be installed automatically

- For Arch Linux, (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
$ sudo pacman -U ./xx.pkg.tar.zst
#+end_src

- For Arch Linux (aarch64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
$ sudo pacman -U ./xx.pkg.tar.zst
#+end_src

- For Fedora Linux (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
$ sudo dnf install ./xx.rpm
#+end_src

- For openSUSE Linux (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
$ sudo zypper install ./xx.rpm
#+end_src

- For Void Linux (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
$ xbps-rindex -a xx-version.x86_64.xbps
$ sudo xbps-install -R ./ xx-version
#+end_src

- For FreeBSD (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
#+begin_src sh
pkg add -f xx.pkg    # As root
#+end_src

- For Windows OS (x86_64/amd64):
Download Chez Scheme 10.0.0 and install it
